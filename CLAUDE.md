# CLAUDE.md

Этот файл содержит инструкции для Claude Code при работе с кодом в этом репозитории.

## О проекте

**ShineBox** — одностраничный сайт мобильной услуги по полировке и детайлингу
автомобилей. Мастер выезжает к клиенту. Сайт двуязычный: **английский (en)** и
**грузинский (ka)** с переключателем языка в шапке.

## Технологии

- **CMS:** WordPress (кастомная тема, без page-builder'а)
- **Окружение:** Docker (`docker-compose.yml` → WordPress + MySQL)
- **Мультиязычность:** плагин **Polylang** (установлен и настроен) — отдельные
  URL `/` (en, по умолчанию, без префикса) и `/ka/`, авто-`hreflang`,
  `<html lang>` переключается. Переключатель в шапке — выпадающий список с
  флагами. Тема также работает автономно через собственный слой переводов
  (`inc/i18n.php`, `?lang=en|ka` + cookie), если Polylang отключить.
- **Front-end:** ванильный HTML/CSS/JS (без сборщика)
- **Валюта:** грузинский лари (₾)

## Запуск (Docker)

```bash
docker compose up -d            # поднять WordPress на http://localhost:8090
# дальше пройти установку WP в браузере, затем:
#   Appearance → Themes → активировать "ShineBox"
docker compose down             # остановить
docker compose down -v          # остановить и стереть БД/данные WP
```

Тема монтируется в контейнер из `./wp-content/themes/shinebox`, поэтому правки
файлов видны сразу без пересборки.

## Ключевые соглашения

- **Тексты UI** живут в `inc/i18n.php` как `key => [ en => ..., ka => ... ]`.
  Выводить только через `shinebox_e('key')` / `shinebox_t('key')` — не хардкодить.
- **Цены и услуги** — единый источник `inc/pricing.php`. И секция «Услуги»,
  и калькулятор читают оттуда, поэтому цены не рассинхронизируются.
- **Калькулятор:** итог = сумма выбранных услуг × множитель типа авто
  (`mult` в `shinebox_car_types()`).
- **Заявки** уходят на email через `wp_mail` (action `shinebox_booking`).
  Адрес почты, телефон, WhatsApp и соцсети настраиваются в
  **Appearance → Customize → ShineBox — Contact & Social**.
- Префикс всех функций/опций — `shinebox_`.
- Экранировать вывод: `esc_html`, `esc_url`, `esc_attr`.
